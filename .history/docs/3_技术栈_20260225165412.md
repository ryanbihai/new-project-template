# TECH_STACK: 简易任务管理系统

## 1. 运行环境 (Runtime Environment)
- **Node.js**: 必须兼容 **v20**，最好兼容 **v16**。
- **Language**: JavaScript 或 TypeScript。**严禁使用 Python**。

## 2. 前端框架 (Frontend Framework)
- **Core**: Vue 3
- **Build Tool**: Vite

## 3. 样式与UI (Styling & UI)
- **CSS Solution**:
  - **推荐**: Tailwind CSS
  - **允许**: CSS Modules, SCSS, Less
- **Component Library**:
  - **Mobile**: Vant UI (允许)
  - **PC**: Element Plus (允许)

## 4. 状态管理 (State Management)
- **Local State**: Vue `ref`, `reactive`
- **Global State**: Pinia (推荐) 或 Vue Context (Provide/Inject)

## 5. 后端/存储
- **Phase 1**: LocalStorage (浏览器本地存储)
- **Phase 2**: Supabase (PostgreSQL + Auth) 或其他后端服务

## 6. 核心依赖清单
- `vue`
- `vite`
- `vant` / `element-plus` (按需)
- `tailwindcss` (推荐)
