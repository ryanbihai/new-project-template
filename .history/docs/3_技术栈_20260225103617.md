# TECH_STACK: 简易任务管理系统

## 1. 前端框架
- **Framework**: Next.js 14.2+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Components**: Shadcn UI (Radix UI)

## 2. 图标库
- **Icons**: Lucide React

## 3. 状态管理
- **Local State**: React `useState`, `useReducer`
- **Global State**: React `Context API` (用于用户偏好，如暗黑模式)

## 4. 后端/存储
- **Phase 1**: LocalStorage (浏览器本地存储)
- **Phase 2**: Supabase (PostgreSQL + Auth)

## 5. 部署
- **Platform**: Vercel

## 6. 核心依赖清单
- `lucide-react`
- `clsx`, `tailwind-merge` (用于 Tailwind 类合并)
- `date-fns` (用于日期处理)
